# Realistic Gun Sound Effects

## Free Sound Sources

To get realistic weapon sounds, download from these free sources:

### 1. Freesound.org (CC0 License)
- **Sniper Rifle**: Search "sniper rifle shot"
  - Example: https://freesound.org/people/Nox_Sound/sounds/554054/
- **Shotgun**: Search "shotgun blast"
  - Example: https://freesound.org/people/InspectorJ/sounds/448226/
- **Rifle**: Search "assault rifle"
  - Example: https://freesound.org/people/nioczkus/sounds/396331/
- **Pistol**: Search "pistol shot"
  - Example: https://freesound.org/people/LeMudCrab/sounds/163456/

### 2. Zapsplat.com (Free with attribution)
- High-quality gun sounds
- Categories: Weapons > Guns

### 3. Mixkit.co (Free)
- https://mixkit.co/free-sound-effects/gun/

## Download Instructions

1. Visit freesound.org
2. Search for weapon type
3. Filter by "CC0" license (no attribution needed)
4. Download as MP3 or WAV
5. Convert to MP3 if needed (smaller file size)
6. Save to `/frontend/sounds/` folder

## Required Files

Save these files in `/frontend/sounds/`:
- `sniper.mp3` - Deep, powerful rifle shot
- `shotgun.mp3` - Loud boom with echo
- `rifle.mp3` - Quick automatic burst
- `pistol.mp3` - Sharp handgun pop
- `headshot.mp3` - Voice saying "HEADSHOT!" (or use text-to-speech)

## File Size Recommendations

- Keep each file under 100KB
- Use MP3 format (better compression)
- Mono audio (not stereo) to save space
- Sample rate: 22050 Hz or 44100 Hz
